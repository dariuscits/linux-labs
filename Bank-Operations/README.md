# Level 1 - Employee Onboarding

## Overview

Welcome to **Level 1** of the Linux Bank Operations Lab.

In this lab, you will practice navigating the Linux file system by locating important documents throughout the bank's server. You will use basic Linux commands to explore directories, read files, and create your own workspace.

## Learning Objectives

By completing this lab, you will practice using:

* `pwd`
* `ls`
* `cd`
* `cat`
* `less`
* `head`
* `tail`
* `mkdir`
* `touch`

## Getting Started

1. Open a terminal.
2. Navigate to this level's folder.
3. Run the setup script:

```bash
python3 create_lab.py
```

This will create the bank directory structure and generate all of the files needed for the lab.

## Your Assignment

After the lab has been created:

1. Open the **bank** directory.
2. Locate the file named **employee_tasks.txt**.
3. Read the instructions carefully.
4. Complete all assigned tasks.
5. Record your answers in **answers.txt**.
6. When finished, verify your work by running:

```bash
python3 verify.py
```

Good luck, and welcome to your first day in the bank's IT department!
