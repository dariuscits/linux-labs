## Linux Bank Operations Lab

# Overview

Welcome back, Bank IT Technician!

After completing your employee onboarding, you have been granted access to assist with basic security tasks.

During a routine security audit, the bank discovered that several confidential files have incorrect permissions. Unauthorized employees may be able to access sensitive banking information.

Your mission is to secure the vault by correcting file permissions and organizing important documents.

---

# Learning Objectives

By completing this lab, you will practice:

- Understanding Linux file permissions
- Reading file permissions with `ls -l`
- Changing permissions with `chmod`
- Copying files with `cp`
- Moving files with `mv`
- Removing files with `rm`
